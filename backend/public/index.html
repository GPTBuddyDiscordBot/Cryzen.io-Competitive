Cryzen Comp Tampermonkey Script

Adds a competitive ranking GUI overlay to Cryzen.io. Tracks kills, deaths, headshots, wins, and losses automatically. View your rank, stats, match history, and leaderboards in-game.

✓ Auto-tracking kills / deaths / headshots
✓ Live rank display with progress bar
✓ Match history with point breakdowns
✓ In-game leaderboard viewer
✓ Manual match submit fallback
✓ Season progress tracking

How to Install

1 Install Tampermonkey browser extension
2 Click the download button below
3 Tampermonkey will open - click Install
4 Go to cryzen.io and press F2 to open the GUI
